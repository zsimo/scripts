    ```sh
    tar -cvf output.tar /dirname
    ```

    ```sh
    echo '{"foo": "lorem", "bar": "ipsum"}' | python -m json.tool
    ```


 - create a symbolic link
     ```sh
     ln -s {/path/to/file-name} {link-name}
     ```   
     ```sh
     ln -s /shared/sales/data/file.txt sales.data.txt
     ```
